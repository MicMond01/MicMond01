## 🚀 What I'm Building

### 🎪 EventShere
A production-grade event management platform connecting planners with venue owners.
- **Real-time** seat layout collaboration with Socket.IO
- **Multi-currency payments** via Paystack, Flutterwave & Stripe (NGN, USD, GBP, EUR + 10 African currencies)
- **3D venue layout planning**, smart guest management, peer reputation system
- **Stack:** React · TypeScript · Node.js · PostgreSQL · MongoDB · Redis · Docker · Cloudinary
